
This repository contains the programs that produce the heartbeat segmentation and alignment in the following paper:
S.S Xu, M.W. Mak and C.C. Cheung, 
"Towards End-to-End ECG Classification with Raw Signal Extraction and Deep Neural Networks",
IEEE Journal of Biomedical and Health Informatics, 2018.

You may run the script to extract heartbeats from raw ECG signal (e.g., record No.: 234).
